### Real-Time Flight Tracker
In this project I focused on a Flight Tracker App using Airlabs API.

##Features Will Be Added
1. Delayed Flights
2. Favourite Flights
3. Notification

## App Contains
1. Protocol Oriented Programming
2. URLSession
3. Programmatic UI
4. MVVM
5. Delegate Pattern
6. Singleton Pattern
7. Unit Tests
